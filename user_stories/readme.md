# AI Prompt Documentation

Prompt to create user stories based on a [technical requirement analysis](../analysis/readme.md).

## Prompts

```text
As IT software analyst and designer, analyse the requirement analysis document to identify key functionalities, including both functional and non-functional user needs. Based on this analysis, generate a set of high-level user stories intended for software developers. Each user story should follow the Cucumber format: "As a [type of user], I want [goal or functionality], so that [value or benefit]." Use a formal tone and ensure that the stories adhere to SCRUM guidelines. The output must be markdown format.
```

Output of the previous prompt is captured in [this document](./user_stories_generated.md).
